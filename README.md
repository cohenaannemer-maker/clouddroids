# CloudDroid 🤖

Your personal APK & file vault — real storage powered by GitHub.

## Deploy to GitHub Pages (5 minutes)

### Step 1 — Create a new repo
1. Go to github.com/new
2. Name it: `clouddroid`
3. Set to **Public**
4. Click **Create repository**

### Step 2 — Upload files
1. Upload ALL files from this ZIP into the repo root
2. Make sure these are all present:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icons/` folder

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** → **/ (root)**
4. Click **Save**
5. Wait ~60 seconds

### Step 4 — Open your app
Your app is live at: `https://COHENAANNEMER.github.io/clouddroid`

### Step 5 — Connect GitHub (first time)
1. Open the app
2. Get a Personal Access Token:
   - Go to github.com/settings/tokens/new
   - Name: CloudDroid
   - Expiration: No expiration
   - Scope: ✅ repo
   - Click Generate token
3. Paste the token into the app
4. Done! Your private vault is created automatically.
